# [05] 키즐링 — 박람회 리드 맞춤 후속 이메일 자동화

트랙: 마케팅 / PM | 작업 시간: 6시간 | 개인 과제
채용 TO: 마케터 1명 · PM 1명

## 기업 배경

키즐링(주)은 알파세대 어린이를 위한 **안전한 어린이 전용 숏폼 챌린지 플랫폼**과 **학습경험플랫폼(LXP)** 을 운영하는 에듀테크 스타트업입니다. 아이들이 영어스피치·줄넘기·그림·음악 챌린지에 숏폼으로 참여하고, AI가 콘텐츠를 추천·평가·필터링합니다(유해 콘텐츠 차단율 99%). **CES 2026 혁신상**, **구글 창구 7기**, **중기부 디딤돌 R&D**에 선정된 성장형 스타트업입니다. B2B 핵심 제품은 **키즐콘(Kizlcon)** — 학교·학원·교육기관을 위한 온라인 대회 접수·결제·심사·결과발표 **원스톱 SaaS**입니다.

## 현재 상황과 문제

키즐링 영업팀은 **에듀플러스위크** 등 교육 박람회에서 수십 명의 명함과 상담 메모를 수집합니다. 그런데 후속 이메일은 **담당자 1인이 전부 수작업**으로 작성합니다.

- **리드 1명당 10~15분** → 50명이면 8시간 이상 소요. 1인이 감당 불가.
- **직책별 맥락 분기 미흡**: 원장은 '원생 효과', 팀장은 '도입이 쉬운가', 사업개발은 '수익 모델'을 묻는데, 같은 톤의 이메일은 누구에게도 꽂히지 않음.
- **상담 내용 휘발**: 부스에서 나눈 대화가 이메일에 반영되지 않아 "우리를 기억 못 하는 회사" 인상.
- **48시간 골든타임 이탈**: 후속이 늦으면 관심이 식고 경쟁사에 기회를 빼앗김.

## 미션

`data/leads_sample.csv` 의 리드 명단을 분석하여:
1. 각 리드에 대해 **직책별 소구점이 반영된 개인화 이메일 초안**을 생성하세요 (원장 / 팀장 / 사업개발).
2. 각 이메일은 ①맞춤 인사 ②부스 상담내용 언급 ③직책별 소구점→제품(키즐콘 기능) 연결 ④명확한 CTA 4요소를 포함하세요.
3. 상담 메모가 비었거나 모호한 리드, 이메일이 없는 리드는 **예외 처리(review_flag)** 하세요.

> 발송은 사람이 검토 후 진행합니다. 이 과제의 범위는 **초안 생성까지**입니다.

## 입력 데이터

`data/leads_sample.csv` — 50건의 박람회 리드 데이터 (원장 20명 / 팀장 18명 / 사업개발 12명)

| 컬럼명 | 설명 |
|--------|------|
| lead_id | 리드 고유번호 (L001~L050) |
| name | 담당자 이름 |
| organization | 소속 기관·기업명 |
| org_type | 학원 / 학교 / 에듀테크기업 / 공공기관 |
| title | 원장 / 팀장 / 사업개발 |
| email | 이메일 주소 (일부 결측) |
| consultation_note | 부스 상담 메모 (일부 결측·모호) |
| interest_feature | 대회운영SaaS / 콘텐츠안전필터 / 파트너십 |
| follow_up_priority | 상 / 중 / 하 (CTA 강도 조절에 활용, **직접 점수 산출 금지**) |

**데이터 품질 주의사항 (정제·예외 처리가 필요합니다):**
- 이메일 결측: 1건 (이메일 없는 리드 → 발송 불가, 별도 후속 메모)
- 상담 메모 결측: 2건 (빈 셀 또는 "상담 메모 없음")
- 상담 메모 모호: 1건 (니즈 파악이 안 되는 메모)
  → 위 결측·모호 케이스는 `review_flag = Y` (사람이 보강 후 발송)
- 완전 중복 리드: 1건 (lead_id만 다르고 내용 동일 → 한 건으로 처리)

## 제출 가이드

**기본 (필수)**
- `output/email_drafts.csv` 또는 `.xlsx`: 리드별 이메일 초안
- 컬럼: `lead_id, subject(직책별 차별화), body(인사→상담언급→제품연결→CTA, 250~400자), review_flag(Y/N)`
- 노션 또는 구글 시트 공개 링크로 제출

**권장**
- 새 리드 CSV에도 동일 품질로 도는 재현 파이프라인 (/명령 또는 스크립트)
- 직책별 소구점·CTA·review_flag 기준을 본인이 직접 설계

**도전**
- 리드 패턴 기반 후속 캠페인 기획안 (세그먼트·근거 리드·후속 시퀀스·우선순위)
- Skill(/명령)로 패키징하여 재현 가능한 형태로 제출
